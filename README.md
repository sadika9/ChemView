ChemView
========

A chemical structure viewer


Dependencies: Qt and Open Babel

Compile: 
````
qmake && make
````
License: GPL

Note:
chemical structures from: [Chemical Structures project](http://chem-file.sourceforge.net/) and [Avagrado project] (http://avogadro.openmolecules.net/)
